Challenge 0: Working/Not Working LED Lights Using the Raspberry Pi and Python

Challenge 1: Once, we knew how to connect the circuits correctly to get the LED lights on, we began coding using the language of python. Brittney and I, primarily began doing research on how we could change the pattern of the lights, in which we did. Once, we found out how to do it, we began coding in which we simply delayed the timing for on and off button by five seconds and had it repeat twice before it turned off.The way we did this was by using "import time", so that when we code "time.sleep()" it will control the flow of our code; using this would allow us to tell the raspberry pi when to turn off and on the LED lights when inputting a time. Then we began setting up the GPIO pins to the corresponding slot, so that it could be used for an input or output. We have to let the RPI know we will be using the GPIO pin 18 as an output (as you can see if you press the Python button on the toolbard.)Then in our code you will see that we have "GPIO.output(18.GPIO.High)", the reason why it is in parenthesis is, because we are selecting the pin number 18 and turning it on writing in High and then when we want the pin to turn off, we would write in "low". Then doing that we inputted "time.sleep(5)", letting the Raspberry pi that we want the light to go off for five seconds, before turning it off. Lastly, at the end of the code it states, "GPIO.cleanup()" basically just end the program and helps avoid causing an damage to our RPI by shorting out the pins.
